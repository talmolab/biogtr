### Development
1. Clone the repository:
```
git clone https://github.com/talmolab/dreem && cd dreem
```
2. Set up in a new conda environment:
```
conda env create -y -f environment.yml && conda activate dreem
```

### Uninstalling
```
conda env remove -n dreem
```
